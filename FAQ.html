Prova prima ad impostare una materia nella pagina 'Materie'
Trovi i passaggi nella home di questo sito o all'interno di ARET alla pagina 'Info'
Succede a volte che i file pdf si possano corrompere
Una soluzione potrebbe essere quella di scaricare nuovamente i file originali
Oppure provare a ricrearli
ARET memorizza le materie già impostate, non devi ricrearle
Viene solo riportato all'inizio l'indice delle domande e delle risposte
Questo perchè ARET genera delle domande casuali ogni volta in modo diverso
Per garantire un ottimo grado di casualità e rendere la tua preparazione superiore
ARET ha bisogno di file con la stessa lunghezza per poter funzionare
In caso contrario le domande e le risposte non coincideranno
Assicurati che i file utilizzati siano di lunghezza uguale



ARET è un software gratuito, non open-source e l'utilizzo si intende così com'è.
altre informazioni sono reperibili nell'EULA

ARET è stato costruito grazie a Python e Customtkinter

La pagina web viene hostata da GitHub grazie a GitHub Pages
